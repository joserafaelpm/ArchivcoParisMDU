Como ingreso al software o Inicio de Sesion

Ingresas tus credenciales que en esta caso son usuario y clave, y el mismo software se encargara de identificar tu rol.

Configuracion ¿Server o cliente?

Configuracion del aplicativo, esto permite que el software se puede conectar los computadores, mediante la red interna de la institución, habra un pc llamado server y todos los demas seran los clientes que se conectaran la ip del Server.

Server

Cliente
